# AI Agent Frameworks Beginner Map

A beginner-friendly static website that compares three AI agent frameworks:

- **LangGraph** — workflow track for reliable, stateful agent processes
- **CrewAI** — project-team model for role-based agent collaboration
- **Microsoft AutoGen** — meeting-room model for multi-agent conversations, with a maintenance-mode caveat

The site now includes both:

1. **Version 1: Original Beginner Map** — concise infographic and comparison table.
2. **Version 2: Visual Beginner Guide** — simple diagrams and decision guidance for beginners.
